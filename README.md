This repository contains experiments and implementations related to Deep Learning.  
Deep Learning is a subset of Machine Learning that uses neural networks with multiple layers to learn patterns from large amounts of data.

# Topics Covered
- Linear Regression
- Neural Networks
- Image Classification
- Model Training and Evaluation
- Data Preprocessing

# Tools and Technologies
- Python
- Google Colab
- NumPy
- Pandas
- Matplotlib
- TensorFlow / PyTorch

# Purpose
The purpose of this repository is to practice and implement Deep Learning algorithms, understand model behavior, and document experiments performed during learning.
